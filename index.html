HTML

  • The head element contains information about the webpage, which is metadata.
  • The body element contains all the things that a user will see on the webpage.
  • HTML is the foundational structure of websites and webpages.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three main CSS application methods, but the most common is to use a style sheet.

Git

  • git status: checks what the current branch is
  • git checkout -b branch-name: creates and switches to a new branch
